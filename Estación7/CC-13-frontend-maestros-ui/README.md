# CC-13 — Frontend Maestros UI (Unidad 3)

**Milestone:** M1 | **Estimado:** 7d | **Prioridad:** medium
**Unidad:** maestros-unidad-3 | **Depende de:** CC-11, CC-08

## Definición
UI de administración para gestionar los 3 catálogos: tablas con CRUD, interface de upload/import, progress bar e historial de cambios.

## Acceptance Criteria
- Tablas con búsqueda, ordenamiento y paginación
- Crear/Editar en modal, Inactivar con confirmación
- Upload de archivo con preview antes de importar
- Progress bar para imports grandes
- Change history viewer por registro
- Solo visible para roles Admin y Manager

## Plan de Tests
- Crear defecto → aparece en tabla
- Inactivar → desaparece de lista activos
- Upload CSV → preview muestra datos correctos
